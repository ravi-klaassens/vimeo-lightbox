Custom Video Element

A lightweight, modular solution for responsive background videos with optional lightbox functionality

How It Works

This custom video element provides a responsive background video with an optional lightbox mode. It's designed to be lightweight, modular, and easy to implement.

Implementation

Simply include the script and use the HTML structure as shown in the examples above.

<script src="https://cdn.jsdelivr.net/gh/ravi-klaassens/vimeo-lightbox@master/custom-video-element.min.js"></script>

Using the data-id attribute (Recommended)

<div class="custom-video-container" data-lightbox="on" data-color="00bcd4" data-id="YOUR_VIDEO_ID">
  <div class="embed-wrapper"></div>
</div>

Using the traditional iframe method

<div class="custom-video-container" data-lightbox="on" data-color="00bcd4">
  <div class="embed-wrapper">
    <iframe src="https://player.vimeo.com/video/YOUR_VIDEO_ID?background=1&autoplay=1&loop=1&byline=0&title=0"
        frameborder="0" allow="autoplay; fullscreen; picture-in-picture" allowfullscreen></iframe>
  </div>
</div>

Parameters

  • data-id - Vimeo video ID (recommended method)
  • data-lightbox - Set to "on" to enable lightbox functionality
  • data-color - Custom color for controls (hex without #)
