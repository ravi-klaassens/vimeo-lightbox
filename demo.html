Custom Video Element

A lightweight, modular solution for responsive background videos with optional lightbox functionality

How It Works

This custom video element provides a responsive background video with an optional lightbox mode. It's designed to be lightweight, modular, and easy to implement.

Background Video Only

Background Mode

No lightbox functionality

Background Video with Lightbox

Lightbox Mode

Click the play button to open lightbox

Implementation

Add the following HTML structure to your page:

<div class="custom-video-container" data-lightbox="on" data-color="00bcd4">
    <div class="embed-wrapper">
        <iframe src="https://player.vimeo.com/video/YOUR_VIDEO_ID?background=1&autoplay=1&loop=1&byline=0&title=0"
            frameborder="0" allow="autoplay; fullscreen; picture-in-picture" allowfullscreen></iframe>
    </div>
</div>

Available Options

  • data-lightbox="on" - Enable lightbox functionality (default is "off")
  • data-color="00bcd4" - Custom color for lightbox controls (use hex color without #)

Features

  • Fully responsive background videos
  • Optional lightbox mode with custom controls
  • Customizable control colors
  • Mobile-friendly design
  • No dependencies (loads Vimeo API only when needed)
